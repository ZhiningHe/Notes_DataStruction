线性表

```
概念：由n个数据元素组成的有限序列，每个元素具有相同结果。
存储方法分为：1.链式表 2.顺序表

逻辑上连续，物理上不一定连续。

class Node{
    Object data;
    Node next;
}
```

```
客户端：
1.不能直接使用node结点，客户端只需要关心数据存储不关心结点关系
因此将node作为内部类隐藏起来

数据存储：
1.头插法
  新节点指向头节点所指的结点
  头节点指向新节点
  size++
2.任意地址插入
  找到插入位置的前驱
  新节点指向插入位置结点
  前驱结点指向新节点
  size++
  
优化点：
由于在任何位置插入结点时需要判断是否是头节点的插入，因为头节点没有前驱结点。
引入一个dummyHead，虚拟头节点。保证每一个结点都有前驱结点。
```
